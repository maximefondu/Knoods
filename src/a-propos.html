Pourquoi ce projet ?

Ce projet est réalisé dans le cadre d’un travail de fin d’année où mes objectifs étaient de monter mes compétences à travers cellui-ci.

J’ai donc réalisé un projet mixant de l'UI et du Javascript étant donné que ces les deux compétences que je veux mettre en avant.

Mes objectifs

Dès l’année prochaine, nous avons pour objectif de trouver un stage pour accroître notre expérience au seing d’une société.

Mon objectif lors de ce stage est d’apprendre toutes les étapes d’un Designer/Developper web and mobile au seing d’une société mais surtout apprendre toutes les astuces des personnes ayant de l’expérience.
